# Taskpad Web Deploy

A static PWA. Deploy to any host that serves files over HTTPS (Cloudflare Pages, Netlify, Vercel, etc.).

## Files to deploy
```
index.html
app.js
app/sync-core.mjs
manifest.json
sw.js
config.json
icon-192.png    <- replace with proper 192x192 px icon
icon-512.png    <- replace with proper 512x512 px icon
fonts.css       <- optional, generated by `node download-fonts.mjs`
fonts/          <- optional, generated by `node download-fonts.mjs`
```

## Optional: offline fonts
Run from the tray project root:
```
node download-fonts.mjs
```
Then copy `src/fonts.css` and `src/fonts/` into this web folder.
Without it the app falls back to Google Fonts and needs internet for fonts.

## config.json
Set your Cloudflare Worker URL here. Defaults to the bundled fallback URL if absent.
```json
{ "workerUrl": "https://your-worker.workers.dev" }
```

## Service worker cache
The service worker caches `index.html`, `app.js`, `app/sync-core.mjs`, and `manifest.json`.
It never caches `config.json` so URL changes take effect immediately.
After deploying a new version, bump the `CACHE` version string in `sw.js`.
